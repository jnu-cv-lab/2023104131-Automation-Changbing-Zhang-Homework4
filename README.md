# 图像下采样、混叠与自适应抗混叠滤波实验

## 项目简介

本项目通过生成棋盘格和密集 Chirp 测试图像，系统地研究了图像下采样过程中混叠现象的产生机理，验证了高斯低通滤波器作为抗混叠预滤波的有效性，并实现了基于局部梯度估计的自适应下采样算法。实验涵盖空间域和频域分析，量化了不同高斯核参数对混叠抑制的影响，并比较了自适应与统一下采样的性能差异。

## 实验内容

1. **生成测试图**：棋盘格（16×16 方块）和径向 Chirp 图（频率从中心 0.01 线性增至边缘 0.495）。
2. **混叠观察**：对比直接下采样与高斯预滤波下采样的空间域及频域结果。
3. **参数优化**：固定下采样倍数 M=4，测试 σ=0.5,1.0,2.0,4.0 下的频谱高频能量，寻找最优 σ 并与理论值 0.45M 比较。
4. **自适应下采样**：根据局部梯度估计下采样因子 M（2~4），对每个块独立滤波采样，并与全局 M=4 下采样对比误差。

## 环境要求

- Python 3.8+
- OpenCV 4.x
- NumPy
- Matplotlib

安装依赖：
```bash
pip install opencv-python numpy matplotlib
```

## 快速开始

1. 将代码保存为 `experiment.py`。
2. 运行程序：
   ```bash
   python experiment.py
   ```
3. 程序将依次显示三幅图像并保存为：
   - `part1_aliasing.png`：第一部分结果（空间域+频谱）
   - `part2_sigma_comparison.png`：不同 σ 下采样结果及频谱
   - `part3_adaptive.png`：自适应下采样结果、误差对比图
4. 控制台将输出不同 σ 下的高频能量及最佳 σ 值。

## 实验结果摘要

### 第一部分：混叠与抗混叠
- **直接下采样**：棋盘格出现摩尔纹，Chirp 图产生低频虚假条纹；频谱中出现混叠产物（十字亮纹或环形折叠）。
- **高斯预滤波**：混叠消失，图像模糊；频谱高频衰减，干净无伪影。

### 第二部分：σ 参数选择
- 理论最优 σ = 0.45×4 = 1.8。
- 实验测得高频能量最小值出现在 σ=2.0 附近（接近理论值）。
- σ 太小→混叠残留；σ 太大→过度模糊。

### 第三部分：自适应下采样
- 自适应方法在边缘高频区使用更小的 M（如 2），保留更多细节。
- 与全局 M=4 下采样相比，自适应方法在边缘区域的误差显著减小（误差差异图红色区域）。
